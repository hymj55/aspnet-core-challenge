# Laptop Catalogue - Myungjeong Han (9031031)

## Overview
A simple web application to manage a laptop catalogue.

## Model Properties
- `Id` (int)  
- `Model` (string)  
- `Brand` (string)  
- `Year` (int)  

## How to Run Each Part
### Part 1: Razor Pages
1. Open `Part1_RazorPages_Laptop` in VS Code.  
2. Run the project.(dotnet run)
3. Home page displays the laptop list.  
4. Click "Add New Laptop" to open the form, fill it, and submit to add a laptop.  

### Part 2: MVC
1. Open `Part2_MVC_Laptop` in VS Code.  
2. Run the project.(dotnet run)
3. Home page displays the laptop list.  
4. Click "Add New Laptop" to open the form, fill it, and submit to add a laptop.  

## Note
- Data is stored in a static in-memory list, so all added laptops reset when the application restarts.
