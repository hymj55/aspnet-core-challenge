## Overview
This project is an ASP.NET Core MVC web application for MJ Line Clothings, showcasing products, a contact form, and a professional site design.
It uses Entity Framework Core with SQLite and demonstrates both scaffolded and manual pages.

## Model Properties
* ClothingId (int, primary key)  
* Name (string)  
* Brand (string)  
* Price (double)  
* Size (string) 

## How to Run
1. Open this project in VS Code.  
2. Run the project with `dotnet run`.  
3. The Home page displays the laptop list.  
4. Click “Add New Clothing” to create a new entry.  
5. Use Edit, Details, and Delete buttons to manage existing Clothings.  

## Note
- Database connection and scaffolding are already set up.  
- Data is stored in SQLite and persists between runs.
